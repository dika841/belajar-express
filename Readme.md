Perintah yang tersedia pada Service ini :

## Langkah petama

### Install Depedencies

```bash
yarn install
```

### Set Environment Variables

### Pull & Generate Db

```bash
yarn prisma generate
```

```bash
yarn prisma db pull
```

### Build

```bash
yarn build
```

## Langkah kedua

### Run service

```bash
yarn dev
```

### Run TypeScript

```bash
yarn ts
```

## Menjalankan prisma studio
